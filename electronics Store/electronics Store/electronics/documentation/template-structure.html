Template structure

See The World With This Section Includes Information On Complete Template And Its Separate Files Structure..

Package structurec

Every Bubble's Passed Its After Html Website Template Package Extracting You'll See The Following Files Structure:.

Documentation - includes template documentation files (on template editing and setup).
  • documentation.txt - includes template documentation files (on template editing and setup).
Site - includes template's pages as .html files.
  • css - includes all .css files, used in template.
  • fonts - includes all custom typography and iconic fonts, used in template.
  • images - includes all images, used in .html files.
  • js - includes JavaScript libraries and jQuery plugins.

HTML Structure

HTML Website Template .html pages structure is built on the following principle:

<!DOCTYPE html>
   <html lang="en">
     <head>
         <!-- Your Stylesheets, Core Scripts (jQuery etc) & Title -->
         ...
     </head>
     <body>
     <div class="page">
         <!-- Page Head -->
         <header class="page-head">
             ...
         </header>

         <!-- Page Content-->
         <section>
                     ...
         </section>

         <!-- Page Footer-->
         <footer class="page-footer">
                 ...
         </footer>
     </div>
     <!-- Javascript -->
     <script src="js/script.js"></script>
     <script src="js/script1.js"></script>
     </body>
   </html>
