Changing the Logo

In Html Website Template, You Can Use Either Custom Image Or Text Logo.

Attention: Html Website Template Is An Html Product. Thus, The Logo Should Be Changed Manually In Every Target .Html File. The Best Part Of The Day.

Changing the Logo

Header logo for the desktop version of the site should be changed in Page Head section of the target page with the .logo class.


        < !-- Logo Start-->
        < div class="header-item-left">
            <a href="./index.html" class="brand">
                < img src='./images/electronics_logo_w.png' alt='logo' />
            </a>
        </div>
        < !-- logo End-->
